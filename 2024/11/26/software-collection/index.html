我的软件收藏集

陈列一些我很喜欢的 PC 端软件 🥰

Scoop

  • A command-line installer for Windows.
  • 链接：官方介绍 | 安装说明 | Wiki
  • 平台：仅 Windows，它由纯 PowerShell 实现
  • 评价：
    • 简易的软件安装工具，尽管它功能非常简单，但还是足够好用了
    • “好用”体现在：不用在安装程序里点来点去，不用操心安装位置，不需要操纵 PATH 环境变量，安装卸载都很干净，纯命令行解决所有问题
    • 格外适合用来 安装/卸载/更新 Windows 系统下的 CLI 程序或者 portable 应用，例如 Git
    • 小缺点是必须能顺畅访问 GitHub，可能不适合非 GitHub 用户使用

Clash Verge Rev

  • A Clash Meta GUI based on Tauri.
  • 链接：官方文档 | GitHub Release 下载
  • 平台：Windows / Linux / MacOS
  • 评价：高知名度的 Clash for Windows 替代品，没有它活不了

Obsidian

  • The private and flexible writing app that adapts to the way you think.
  • 链接：官方介绍 | 官方教程中英 | 中文论坛
  • 平台：PC / 移动端全平台
  • 评价：
    • 集高性能、高定制化、隐私安全为一体的笔记工具，它几乎可以成为任何你想的样子
    • Obsidian 是我读研之后才开始使用的，真的非常后悔它没有在我读本科或者高中的时候用它，因为它会成为人类在数字世界里生活方式的一部分

Typora

  • A truly minimal markdown editor.
  • 评价：曾经是一款非常亮眼的所见即所得 markdown 编辑器，现在已经基本被 Obsidian / VSCode 代替了。以前用了几年，还氪金支持了，现在留着当 .md 文件的默认打开应用

Windows Terminal

  • 微软的新一代终端工具
  • 链接：官方介绍
  • 平台：仅 Windows
  • 评价：
    • 如果说 Ubuntu 的自带终端丑的还能接受，那么 Windows 自带的命令提示符终端是真的丑到辣眼睛 —— 微软的这款终端确实非常令人满意，轻量化、可定制、纯 JSON 配置，也可以满足一些 SSH 客户端需求
    • 可以根据微软文档的教程，把 Windows Terminal 设置成系统默认终端，这样就可以永久告别辣眼睛的命令提示符了

Tabby Terminal

  • A terminal for the modern age.
  • 链接：官方介绍 | GitHub 项目
  • 平台：Windows / Linux / MacOS
  • 评价：
    • 基于 Electron/Typescript 的跨平台终端，比 Windows Terminal 更现代、功能更丰富，也同样开源
    • 对我来说，它比 Windows Terminal 的优势在于多了 SSH client 支持和简易的 SFTP，定制化自由度也更高，还有一个toggle-window 的全局快捷键非常方便
    • 劣势也很明显，毕竟是 Electron，磁盘和内存开销确实远高于 Windows Terminal

Zotero

  • A free, easy-to-use tool to help you collect, organize, annotate, cite, and share research.
  • 链接：官方介绍 | 浏览器插件
  • 平台：Windows / Linux / MacOS
  • 评价：
    • 文献工具，配合浏览器一键保存，一键导出 BibTeX 文献列表，非常丝滑
    • 可以配合 WebDAV 云同步 就可以在不同设备随时看文献

VS Code

  • 最强代码编辑器 / IDE ，没什么好解释的，除了各类开发工作，还可以用来写 markdown, latex

Nutstore 坚果云

  • 国内用着方便省心且不用花钱的同步盘，提供 WebDAV
  • 它的 WebDAV 功能配合 Zotero 之类的应用，能实现自动云同步
  • 免费版真的太够用了，丝滑省心

一些浏览器插件

  • 为什么你们就是不能加个空格呢？

    • 自动在网页中所有的中文字和半形的英文、数字、符号之间插入空白
    • 链接：Chrome 插件 | GitHub 仓库 pangu.js
    • 没错，这个插件的名字就叫“为什么你们就是不能加个空格呢？” 。懂的人自然懂，不懂的也没必要哈哈哈，喜欢作者的个性~

      汉学家称这个空白字元为「盘古之白」，因为它劈开了全形字和半形字之间的混沌。另有研究显示，打字的时候不喜欢在中文和英文之间加空格的人，感情路都走得很辛苦，有七成的比例会在 34 岁的时候跟自己不爱的人结婚，而其余三成的人最后只能把遗产留给自己的猫。毕竟爱情跟书写都需要适时地留白。

  • SwitchyOmega

    • 链接：Chrome 插件 | GitHub 仓库
    • 这是一个浏览器页面的代理设置工具。它允许你控制不同网页使用何种网络代理。你可能想问这和 Clash rules 有什么区别，它的区别在于让浏览器主动访问某个你指定的代理地址，而不是让 Clash 作为你的系统代理并决定每个连接。
    • 比如当你希望某些网站走代理，而某些网站直连时，这个插件设置起来就非常方便
    • 不过说到底，是否使用系统代理也只是个人偏好，我比较喜欢控制的细一点

一些字体 & 配色

  • Nerd Fonts
    • 链接：官网
    • 带图标字形、连字的字体集，IDE / terminal 专用，开发者必备。我喜欢这两个字体：
    • Cascadia Code 原本是微软主推的字体，被 Nerd Fonts 增强后更好用了
    • Fira Code 真的很好看……
  • DejaVu Math TeX Gyre
    • 链接：GitHub 项目
    • 这是一款英文衬线字体。它的字母相对更饱满（大概就是比其他字体更宽一些），我觉得它的主要优点就是易于阅读
  • One Dark Pro
    • 我的 IDE 比较喜欢的颜色主题，可以算是对比度比较高的一款了，颜色很鲜艳而且背景足够深，视觉效果很好
  • Catppuccin
    • 链接：官网
    • GitHub 上收获 15k stars 的配色方案，色彩鲜活且柔和，对比度适中
    • 可以用于设计，也可以作为颜色主题使用
    • 系列里的 Latte 是 light theme，而 Mocha 是 dark theme，另外另个是 Mocha 的低对比度变体，可以先从 Latte 和 Mocha 里面选
